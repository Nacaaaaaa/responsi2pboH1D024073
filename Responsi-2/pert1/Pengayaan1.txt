Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Jelaskan perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek.
2. Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya?
3. Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.
4. Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

Jawaban:

1. Perbedaan antara fungsi dan prosedur dalam PBO adalah fungsi melakukan perhitungan dan mengembalikan nilai kepada pemanggilnya, sedangkan prosedur melakukan serangkaian dan tidak mengembalikan nilai.
2. Penting menggunakan konstanta dalam suatu kelas karena menjamin keamanan data agar nilai standar tidak dapat diubah. Keuntungannya adalah meningkatkan keterbacaan dan mempermudah pemeliharaan kode.
3. Yang terjadi jika mencoba mengubah nilai konstanta setelah objek dibuat adalah compiler Java akan menolaknya dan menghasilkan error. Contoh: mencoba p.faktorAktivitas = 1.5; menghasilkan error 'cannot assign a value to final variable faktorAktivitas'.
4. Cara mendefinisikan konstanta dalam bahasa Java adalah menggunakan kata kunci final. Untuk konstanta milik kelas, biasanya digunakan public static final.
